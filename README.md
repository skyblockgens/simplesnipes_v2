### What it does:

Ah flipping :|

### How to use it?
- `/ss` - Shows the config GUI
- `/ss help` - Shows the help message containing all the subcommands and their usage
- `/ss toggle` - Enables the mod to return a list of suggested items to flip whenever one is found.
